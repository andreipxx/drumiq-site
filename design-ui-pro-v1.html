IMPLEMENTATION NOTES

Typography: Inter for UI text, JetBrains Mono for numbers/data. Load via Google Fonts or bundle. Hero numbers at 48px weight 900, primary labels 10-13px weight 700, micro labels 8px weight 700 with 2-3px letter spacing.

Overlay (CRITICAL): The verdict symbol (44px circle) and RON/km value (20px mono bold) must be readable from 50cm at a glance in under 0.5 seconds. The colored top bar (4px) provides instant color coding. Glow effect on verdict circle reinforces the signal. Keep contrast maximum -- verdict color on near-black background.

Animations to implement:
-- Splash: logo breathing glow (3s ease), progress bar shimmer
-- Overlay appear: slide-in from left + fade (200ms cubic-bezier)
-- Verdict circle: subtle pulse glow on appear (glowPulse keyframe)
-- Cards: fadeInUp on mount (staggered 50ms per card)
-- Progress bar: smooth width transition (600ms ease)
-- Tab switch: cross-fade content (150ms)
-- Pull-to-refresh: custom green spinner

Glassmorphism: Use backdrop-filter: blur(10-20px) on overlay card and bottom nav. Background opacity 0.94-0.96 for readability. Border 1px rgba(255,255,255,0.05) for glass edge.

Gradients: Green accent gradient (accentDeep to accent, 135deg) for primary buttons and active states. Radial glow gradients on hero cards. Never flat solid green -- always gradient or glow.

Spacing system: 4/8/12/16/20/24/32px scale. Card padding 12-16px. Screen horizontal padding 16px. Gap between cards 8px. Section margin-bottom 16px.

Bottom navigation: 5 tabs with SVG stroke icons (20px). Active state: accent color + glow + top indicator line (2px). Backdrop blur for depth. Home padding accounts for safe area (24px bottom).

One-handed use: All primary actions in bottom 60% of screen. Overlay CTA is large touch target (56px height). Bottom sheet pattern for secondary settings. Tab buttons have min 44px touch target. Verdict list items have generous padding.
